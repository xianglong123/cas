### XPath 常用表达式

| 表达式 | 含义 | 
| :-----| :---- | 
| nodename | 选取指定节点的所有子节点 | 
| / | 从根节点选取指定节点 | 
| // | 根据指定的表达式，在整个文档中选取匹配的节点，这里并不会考虑匹配节点在文档中的位置 | 
| . | 选取当前节点 | 
| .. | 选取当前节点的父节点 | 
| @ | 选取属性 | 
| * | 匹配任何元素节点 | 
| @* | 匹配任何属性节点 | 
| node() | 匹配任何类型的节点 | 
| text() | 匹配文本节点 | 
| ｜ | 选取若干个路径 | 
| [] | 指定某个条件，用于查找某个特定节点或者包含某个指定值的节点 |
